此App基于APICloud平台，使用其中的自定义Loader(官网有教程说明)

# 项目启动：需安装webpack、gulp、mongodb

1. npm install
2. 本地先启动mongodb
3. gulp server(启动webpack和本地服务器)
4. 启动wifi真机同步服务，手机ip和电脑ip一致
5. 在IPass_APP/config.xml文件目录下，使用wifi真机同步，即可在手机端查看运行的App
